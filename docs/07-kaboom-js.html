Flappybird mit Kaboom.js

Ein grosses Thema in der Informatik ist die Spielentwicklung. Dafür können verschiedene Game-Engines verwendet werden. Wir schauen uns mal eine sehr einfache Game-Engine an. Damit können ganz einfache Spiele in Javascript geschrieben werden und im Browser gespielt werden. Sie brauchen also nur den Link zu Ihrer Webseite zu verteilen, und alle können Ihr Spiel spielen.
